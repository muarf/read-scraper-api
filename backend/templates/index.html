🚀 API Scraper - Documentation

Important: Vous devez d'abord créer une clé API en visitant /init

Configuration

Toutes les requêtes API nécessitent un header X-API-Key contenant votre clé API.

Endpoints Publiques

POST /api/v1/scrape

Créer un nouveau job de scraping

{
    "url": "https://exemple.com/article"
}

Response:
{
    "job_id": "abc123...",
    "status": "pending",
    "url": "https://exemple.com/article"
}
GET /api/v1/job/{job_id}

Obtenir le statut d'un job

GET /api/v1/article/{article_id}

Obtenir un article par son ID

GET /api/v1/article/{article_id}/pdf

Télécharger le PDF d'un article

GET /api/v1/articles?limit=50&offset=0&search=...&site_source=...

Lister les articles avec pagination et filtres

GET /api/v1/search?q=query

Rechercher dans les articles

Endpoints Admin

GET /api/v1/admin/stats

Obtenir les statistiques globales

GET /api/v1/admin/articles

Lister tous les articles (admin)

DELETE /api/v1/admin/article/{article_id}

Supprimer un article

GET /api/v1/admin/jobs

Lister tous les jobs

POST /api/v1/admin/apikeys

Créer une nouvelle clé API

GET /api/v1/admin/apikeys

Lister les clés API

DELETE /api/v1/admin/apikeys/{api_key_id}

Révoquer une clé API

POST /api/v1/admin/cleanup

Nettoyer les données anciennes

Flux complet

  1. Créer un job avec POST /api/v1/scrape
  2. Polling sur GET /api/v1/job/{job_id} jusqu'à status=completed
  3. Récupérer l'article_id du job
  4. Obtenir l'article avec GET /api/v1/article/{article_id}
  5. Télécharger le PDF avec GET /api/v1/article/{article_id}/pdf
